# IQToolkit IQAI Service

This service provides AI-powered explanations and chat capabilities for the IQToolkit platform.
It acts as a "Copilot" for the Analyzer, using LLMs to interpret analysis results.

## Tech Stack
- **Language:** Python 3.11+
- **Framework:** FastAPI
- **LLM Client:** pydantic-ai
- **Cache:** Valkey (Redis compatible)

## Setup
```bash
pip install -e .
```
