# IQToolkit Hub

The central orchestration gateway for the IQToolkit platform.
It composes workflows between the Analyzer and IQAI services.

## Tech Stack
- **Language:** Python 3.11+
- **Framework:** FastAPI
- **Database:** PostgreSQL (SQLAlchemy + Alembic)
- **HTTP Client:** HTTPX

## Setup
```bash
pip install -e .
```
